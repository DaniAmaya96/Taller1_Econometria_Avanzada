Pasos para correr el paquete reproducible del Taller 1.

El paquete reproducible asociado se encuentra en el repositorio de GitHub del estudiante Daniel Amaya. https://github.com/DaniAmaya96/Taller1_Econometria_Avanzada.

A continuación, se presenta una lista de los pasos necesarios en caso de que se quiera trabajar localmente con el paquete reproducible:

1. Entrar al siguiente link en el explorador de su preferencia: https://github.com/DaniAmaya96/Taller1_Econometria_Avanzada. En el link encontrará el repositorio con todos los archivos necesarios para correr el código asociado a la resolución del Taller 1.
 

2. Para descargarlo, haga clic en el botón “<> Code” que encontrará en la parte superior. En el menú desplegable, le dará la opción de descargar todos los archivos en formato .zip con la opción “Download ZIP”.
 

3. Elija la ruta de su preferencia y descargue el paquete reproducible.
 

4. Una vez descargado, vaya a su explorador de archivos y descomprima la carpeta.
 

5. Con los archivos descomprimidos podrá abrir localmente el archivo “index.qmd”, en este archivo encontrarán el código markdown con el que fue desarrollado el documento PDF, que incluye las secciones de código en R para los puntos que lo requerían.
 

Notas:

• El archivo en formato .qmd permite abrir leer todas las bases de datos requeridas para la ejecución del código.
 

• El archivo en formato .qmd permite la renderización a formatos .html y .pdf
